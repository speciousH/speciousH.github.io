个人博客搭建指南

前言

作为信息时代的一份子, 你的日常笔记或是随笔书写在哪里? 你是否在查阅资料时无意中访问过这样一些名为个人博客的网站, 这里囊括了美观的外表展示以及丰富的资料储备, 看着这些网站, 你一定心动了吧!

不用担心, 本文将会从零开始, 使用 windows10 操作系统, 一步步搭建一个基于 hexo 和 github 的个人博客网站.

欢迎浏览白嫖!

配置 GitHub

  1. 创建仓库

    如果此前没有 github 账号, 则需要先注册一个.

    新建一个名为你的用户名.github.io的仓库(必须是你的用户名，其它名称无效)

  2. 配置 SSH 免密登录

    在我的配置过程中发现, 使用用户名和密码登录时, 总会爆出各种错误, 同时由于 github 的外网性质, 国内访问时速度慢, 也会在一定程度降低登陆成功的几率, 因此, 配置免密登录会更加方便省事一点, 同时也会更加安全.

    操作步骤:

    1. 首先打开电脑文件夹, 找到 C:\Users\用户名\ .ssh 文件夹并删除;

    2. 在 C:\Users\用户名文件夹下右键打开 Git Bash Here 输入命令

      生成.ssh秘钥, 输入后连敲三次回车, 出现 The Key`s randomart image is: (方框), 说明成功;

    3. 最终生成了一个新的 C:\Users\用户名\ .ssh文件夹, 打开这个文件夹, 找到 .ssh\id_rsa.pub 文件, 记事本打开并复制里面的内容;

    4. 打开你的 github 主页, 进入个人设置 -> SSH and GPG keys -> New SSH key，把复制的内容粘贴进去, title 随便填, 保存即可, 公钥就添加成功了;

    5. 检测是否设置成功, 输入命令

      如果提示Are you sure you want to continue connecting (yes/no)? 输入yes, 然后会看到:

      Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access.

      看到这个信息说明 SSH 已配置成功!
